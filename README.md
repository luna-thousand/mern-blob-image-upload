# ❗ Message 

Don't forget to edit the .env file in Frontend and Backend.

## 🍔 Installation

```bash
git clone https://github.com/luna-thousand/mern-blob-image-upload.git
```

## 🥨 Install Frontend Package & Start

```bash
cd Frontend 
npm i
npm run dev
```

## 🍟 Install Backend Package & Start

```bash
cd Backend 
npm i
npm start
```
